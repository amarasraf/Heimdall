Bebbi perlukan 'API Key' dari EasyParcel untuk panggil abang posmen datang rumah ambil barang.

Langkah-langkah:
  1. Buka Dashboard EasyParcel (tekan butang di bawah & pastikan dah login).
  2. Tekan butang menu (garis tiga ≡) di penjuru atas kiri.
  3. Cari dan tekan menu Integration (ada ikon sambungan).
  4. Pilih sub-menu Add Ecommerce App.
  5. Scroll ke bawah sekali dan pilih kotak API Key / Custom API.
  6. Copy kod panjang tu dan tampal kat kotak atas ni.

*Nota: Kalau kotak API Key TIADA, maknanya EasyParcel tak bagi untuk akaun 'Personal'. Makcik boleh abaikan ni dan terus guna butang 'Download Excel' kat bawah ya!
